# Xcode

###### zenburn.xccolortheme

```bash
mkdir -p ~/Library/Developer/Xcode/UserData/FontAndColorThemes
cp zenburn.xccolortheme ~/Library/Developer/Xcode/UserData/FontAndColorThemes
```
